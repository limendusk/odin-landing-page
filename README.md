# odin-landing-page
A "landing page" web page consisting of a header, 4 different sections, and a footer.

The content of the web page is centered around the Japanese Rock duo Yorushika (ヨルシカ). The web page has a header with links to the duo's official site, merch store, and discography. The 4 main sections contain a brief introduction to the Yorushika, a collection of their albums released so far, a quote from one of their songs, and an invitation for the user to check out their official YouTube channel.

This project was built as a learning exercise to practice using common CSS properties and the Flexbox layout model.
